# CARA — Ecommerce Website
A fully responsive frontend ecommerce website template built with HTML and CSS. CARA is a fashion/clothing store template featuring product listings, seasonal promotions, newsletter signup, and multi-page navigation.

### ✨ Features
- 🏠 Multi-page layout — Home, Shop, Blog, About, Contact
- 🎯 Hero banner with promotional offer ("Up to 70% Off")
- 🛒 Featured Products section — Summer Collection grid
- 🆕 New Arrivals section — Spring/Summer product grid
- 🏷️ Promotional banners — "Buy 1 Get 1 Free", Seasonal Sale, New Footwear Collection
- 🧰 Feature highlights — Free Shipping, Online Order, Save Money, Promotions, Happy Sell, 24/7 Support
- 📧 Newsletter email subscription section
- 📱 App download links — App Store & Google Play
- 💳 Secured payment gateway badges
- 🔗 Footer with Contact info, About links, My Account links, and social media


### 🗂️ Pages
| Page | Description |
|---|---|
| `index.html` | Homepage — Hero, Features, Products, Banners, Newsletter |
